# Linear Regression from Scratch

Predicting house prices based on square footage using pure Python and gradient descent.

---

## What it does

- Fits a line: `y = mx + b`
- Learns using gradient descent
- Plots:
  - Training data
  - Initial guess (light gray)
  - Final line (black)
